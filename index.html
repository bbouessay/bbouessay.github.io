Résumé HTML5

Le HTML sert à struturer le contenu sites. Le HTML5 est la dernière version de HTML. Cette version a été finalisée en octobre 2014 par le W3C.

Certains navigateurs ne supportent pas toutes les fonctionnalités par le HTML5. Afin d'obtenir plus d'information, il est possible d'aller sur le site http://caniuse.com.

Les balises fonctionnent avec des chevrons (" < " et " > "). Exemple : <Exemple> Texte </Exemple>.

Structure minimale d'un document HTML

          <!DOCTYPE html> "Pour indiquer la version de l'HTML"
            <html>
              <head> "Pour indiquer les instructions"
              <meta charset="utf-8" /> "Pour indiquer le jeu d'encodage"
                <title>"nom du titre"</title>
              </head>
              <body>
                  "pour le corp du texte"
              </body>
            </html>

Exemple de balises

  • <strong>Gras</strong>
  • <mark>Surlignage</mark>
  • <em>Italique</em>
  • <h1>Titre 1</h1>, <h2>Sous-titre 2</h2>......jusqu'à......<h6>Sous-titre 6</h6>
  • Pour écrire les < et > : "& l t" (sans espace) pour < et "& g t" (sans espace) pour >
  • Pour garder la même mise en forme que sur le script :

    <pre>

    <code>

    </code>

    </pre>

  • <ol>Liste ordonnée :

    <li> Liste 1 </li>

    <li> Liste 2 </li>

    </ol>

  • <ul>Liste désordonnée :

    <li> Liste 1 </li>

    <li> Liste 2 </li>

    </ul>

  • <dd>Liste définition :

    <li> Liste 1 </li>

    <li> Liste 2 </li>

    </dd>
